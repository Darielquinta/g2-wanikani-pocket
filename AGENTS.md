\# Project instructions



This is an Even Realities G2 web app using Vite, TypeScript, and @evenrealities/even\_hub\_sdk.



Use Node.js 20 or 22+.



Important commands:

\- Install dependencies: npm install

\- Run dev server: npm run dev

\- Build/check: npm run build



When editing:

\- Keep the app compatible with the Even G2 WebView.

\- Prefer simple, readable UI.

\- Do not assume hardware access inside Codex Cloud.

\- Validate changes with npm run build.

\- Use @evenrealities/even\_hub\_sdk for bridge, display, events, and storage.
